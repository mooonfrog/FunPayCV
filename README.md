# FunPayCV
~ Почти тот же FunPayCardinal, только для ВК :)

#### Почему мы (yusxe и pradampx) решили создать этот репозиторий?
- На фоне замедления Telegram - чтобы не терять доступа к FunPay и упрощения его работы (данные с cardinal можно легко перенести в нашего бота) мы решили сделать бота для FunPay.com на платформу ВКонтакте.

## Установка:
```bash install.sh```

## Плагины
~ Плагины поддерживаются так и от FunPayCardinal так и самопис от FunPayCV
